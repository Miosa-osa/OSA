# Prime: OSA Terminal

## Stack
- Next.js 14 (App Router)
- TypeScript strict
- Tailwind CSS + shadcn/ui
- Zustand + React Query

## UI Requirements
- Desktop-first (min 1024px, optimal 1440-1920px)
- Light mode default
- NO emojis - Lucide icons only
- Silver orb branding
- Draggable window system

## Windows
- Agent Chat (main interaction)
- Terminal (live demo)
- Files (file tree)
- Preview (app preview)
- Editor (code editing)

## Agent
Use osa-terminal agent for all OSA Terminal work.
