# Brainstorming Skill

## When This Activates
Before any new implementation or when facing complex decisions.

## Process
1. **Clarify** - What exactly are we solving?
2. **Explore** - Generate 3-5 different approaches
3. **Evaluate** - Pros/cons/trade-offs of each
4. **Decide** - Select best approach with reasoning
5. **Plan** - Break into concrete, actionable steps

## Output Format
```markdown
## Brainstorm: [Topic]

### Problem Statement
[Clear definition]

### Options Considered
1. **[Option A]**: [Description]
   - Pros: 
   - Cons: 

2. **[Option B]**: [Description]
   - Pros: 
   - Cons: 

### Decision
[Selected option] because [reasoning]

### Action Plan
1. [Step 1]
2. [Step 2]
```
